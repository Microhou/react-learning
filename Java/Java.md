# Java 入门

## 什么是类
- student 类

```java
    public class Student {
        String name;
        int age;
        String gender;
        
        void study() {
            System.out.println("Student " + name + " is studying.");
        }
        void play() {
            System.out.println("Student " + name + " is playing.");
        }
    } 
```
